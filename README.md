## 📝 Problem File Template

Each problem is saved under `problems/easy/` (or medium/hard) as `XXXX-problem-name.md`:

```markdown
# XXXX. Problem Title

- **Difficulty:** Easy / Medium / Hard
- **Pattern:** e.g. Hash Map, Sliding Window
- **Date:** YYYY-MM-DD
- **Attempts before solving:** N
- **Time spent:** N min

## Approach
Solve in Python(3) first (fastest for logic), then re-implement the same problem in C++ or JS the next attempt.
Minimize the use of hints and editorial. After used, review the anatomy of each code (Python3, JS, and C++)
```
